# ImGui ESP-IDF Component

[Dear ImGui](https://github.com/ocornut/imgui) packaged as an ESP-IDF component, with a software-renderer backend for `esp_lcd` panels.

- Bundles the unmodified upstream Dear ImGui library
- Uses a software rasterizer written by [LAK132](https://github.com/LAK132)
- Works with any LCD supported by `esp_lcd` component
- Can be tested on the host with QEMU

## Usage

Add the component to your project's `idf_component.yml`:

```yaml
dependencies:
  igrr/imgui:
    version: "^1.92.6"
```

Initialize the port in your application:

<!-- code_snippet_start:examples/imgui_reload/main/imgui_reload_main.cpp:r/^#include "imgui\.h"/:r/^#include "app_ui\.h"/ -->

```cpp
#include "imgui.h"
#include "imgui_port_esp_lcd.h"
```

<!-- code_snippet_end -->

<!-- code_snippet_start:examples/imgui_reload/main/imgui_reload_main.cpp:r/ImGui port/:r/imgui_port_init/+ -->

```cpp
    /* ---- ImGui port ---- */
    imgui_port_renderer_handle_t renderer = nullptr;
    ESP_ERROR_CHECK(imgui_port_new_renderer_rgb888(&renderer));

    const imgui_port_cfg_t port_cfg = {
        .panel_handle  = lcd_handles.panel,
        .width         = BSP_LCD_H_RES,
        .height        = BSP_LCD_V_RES,
        .render_buf    = nullptr,
        .renderer      = renderer,
    };
    ESP_ERROR_CHECK(imgui_port_init(&port_cfg));
```

<!-- code_snippet_end -->

<!-- code_snippet_start:examples/imgui_reload/main/imgui_reload_main.cpp:r/Render loop/:r/imgui_port_render/+ -->

```cpp
    /* ---- Render loop ---- */
    while (true) {
        imgui_port_new_frame();
        app_ui_draw();
        imgui_port_render();
```

## Custom Fonts

Use the `imgui_font_add()` CMake function to embed a TTF font into your firmware:

```cmake
imgui_font_add(${IMGUI_FONTS_DIR}/Karla-Regular.ttf)
```

This compresses the font at build time and generates a header (`Karla_Regular.h`) with the data. Load it before the first frame:

<!-- code_snippet_start:test_apps/qemu_test/main/imgui_main.cpp:r/Compressed font/:r/Karla_Regular\.h/+ -->

```cpp
/* Compressed font generated by imgui_font_add() in CMakeLists.txt */
#include "Karla_Regular.h"
```

<!-- code_snippet_end -->

<!-- code_snippet_start:test_apps/qemu_test/main/imgui_main.cpp:r/Load a custom font/:r/Karla_Regular_compressed_size/+ -->

```cpp
    /* Load a custom font (Karla-Regular at 18px) instead of the default.
     * Create the ImGui context before imgui_port_init() so the font is
     * included when the port builds the font atlas. */
    ImGui::CreateContext();
    ImGuiIO &io = ImGui::GetIO();
    io.Fonts->TexDesiredFormat = ImTextureFormat_Alpha8;
    io.Fonts->AddFontFromMemoryCompressedTTF(
        Karla_Regular_compressed_data, Karla_Regular_compressed_size, 18.0f);
```

<!-- code_snippet_end -->

You can optionally specify a custom symbol name:

```cmake
imgui_font_add(fonts/MyFont.ttf SYMBOL my_font)
# Generates my_font.h with my_font_compressed_data / my_font_compressed_size
```
